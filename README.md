# 📊 E-commerce Sales Dashboard

Interactive dashboard built with Streamlit for e-commerce sales analysis.

---

## 🚀 Demo

https://sql-e-commerce-data-analysis-mssb4a8akeyubhsgmyvksa.streamlit.app/

---

## 📌 Features

* Total revenue KPIs
* Number of orders
* Monthly sales analysis
* Top-selling products
* Revenue by country

---

## 🛠️ Technologies

* Python
* Streamlit
* Pandas
* Plotly

---

## 📂 Project Structure

```
├── app.py
├── requirements.txt
└── data/
    └── train.csv
```

---

## ▶️ How to Run Locally

```
pip install -r requirements.txt
streamlit run app.py
```
