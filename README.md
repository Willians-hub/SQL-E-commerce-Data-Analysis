# 📊 E-commerce Sales Dashboard

Dashboard interativo desenvolvido com Streamlit para análise de vendas de e-commerce.

---

## 🚀 Demo

https://sql-e-commerce-data-analysis-mssb4a8akeyubhsgmyvksa.streamlit.app/

---

## 📌 Funcionalidades

* KPIs de receita total
* Número de pedidos
* Vendas mensais
* Top produtos
* Receita por país

---

## 🛠️ Tecnologias

* Python
* Streamlit
* Pandas
* Plotly

---

## 📂 Estrutura

```
├── app.py
├── requirements.txt
└── data/
    └── train.csv
```

---

## ▶️ Como rodar localmente

```
pip install -r requirements.txt
streamlit run app.py
```
